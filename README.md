# کتابیں - آن لائن کتاب پڑھنے کی ایپلیکیشن

ایک جامع کتاب پڑھنے کی ایپلیکیشن جو آن لائن کتابیں پڑھنے کے لئے ڈیزائن کی گئی ہے۔

## خصوصیات

- سبز اور سفید تھیم
- لاگ ان کی ضرورت نہیں
- 300 صفحات کا مواد تین حصوں میں تقسیم:
  1. حصہ اول: صفحہ 1-100
  2. حصہ دوم: صفحہ 101-200
  3. حصہ سوم: صفحہ 201-300
- تمام آلات پر کام کرنے والا ریسپانسیو ڈیزائن
- کتاب کے صفحات کے درمیان آسان نیویگیشن کے لئے پیجنیشن
- آرام سے پڑھنے کے لئے اسکرول کرنا

## فائل کی ساخت

```
├── index.html              # مرکزی ہوم پیج
├── section1.html           # کتاب کے صفحات 1-100
├── section2.html           # کتاب کے صفحات 101-200
├── section3.html           # کتاب کے صفحات 201-300
├── assets/
│   ├── css/
│   │   └── style.css       # مرکزی سٹائل شیٹ
│   ├── js/
│   │   ├── script.js       # مرکزی پیج کے اسکرپٹس
│   │   ├── section1.js     # حصہ اول کے اسکرپٹس (صفحات 1-100)
│   │   ├── section2.js     # حصہ دوم کے اسکرپٹس (صفحات 101-200)
│   │   └── section3.js     # حصہ سوم کے اسکرپٹس (صفحات 201-300)
│   └── images/
│       └── placeholder.txt # تصویر پلیس ہولڈر
```

## استعمال کا طریقہ

1. `index.html` کو اپنے ویب براؤزر میں کھولیں تاکہ مرکزی پیج دیکھ سکیں
2. نیویگیشن مینو کا استعمال کرتے ہوئے ہر حصے پر جائیں:
   - حصہ اول: صفحات 1-100
   - حصہ دوم: صفحات 101-200
   - حصہ سوم: صفحات 201-300
3. ہر حصے کے نچلے حصے میں موجود پیجنیشن کنٹرول کا استعمال کرکے صفحات کے درمیان منتقل ہوں

## کتاب کے صفحات کو شامل کرنا

اپنی کتاب کے صفحات کی تصاویر کو شامل کرنے کے لئے:

1. اپنی کتاب کے صفحات کی تصاویر کو `assets/images/` فولڈر میں رکھیں
2. اپنی تصاویر کو ترتیب کے مطابق نام دیں:
   - حصہ اول کے لئے: `page1.jpg`, `page2.jpg`, ..., `page100.jpg`
   - حصہ دوم کے لئے: `page101.jpg`, `page102.jpg`, ..., `page200.jpg`
   - حصہ سوم کے لئے: `page201.jpg`, `page202.jpg`, ..., `page300.jpg`
3. تعاون یافتہ فارمیٹس: JPG, PNG, GIF

## براؤزر کی معاونت

یہ ویب سائٹ تمام جدید براؤزرز پر کام کرتی ہے بشمول:
- Chrome
- Firefox
- Safari
- Edge

## نوٹس

- ایپلیکیشن کتاب کے صفحات کو تصاویر کے طور پر دکھانے کے لئے ڈیزائن کی گئی ہے
- تمام سٹائلنگ صرف CSS کے ساتھ کی گئی ہے (کوئی بیرونی لائبریریز نہیں)
- تمام فعالیت صرف وینیلا جاوا اسکرپٹ کے ساتھ نافذ کی گئی ہے (کوئی بیرونی لائبریریز نہیں)
- تصاویر موجودہ صفحے کے نمبر کی بنیاد پر متحرک طور پر لوڈ ہوتی ہیں
